Installation Guide
1
Press Win + R, type %appdata%/.minecraft/resourcepacks and press Enter.
2
Drag and drop the downloaded .zip file into that folder.
3
In-game, go to Options > Resource Packs and move the pack to the "Selected" column.
!
If the game says Pack is outdated, just ignore it. If you have any issues, feel free to ask for help!
